Assumptions & equation used

This simple simulator assumes:

  • Initial daughter atoms = 0 (all atoms start as parent).
  • The system remains closed (no parent/daughter gain or loss).

We use the radioactive decay law:

N/N0 = (1/2)t/t1/2

Solving for time:

t = t1/2 · log2(N0/N)

In this app, N/N0 is the parent fraction remaining (1 − daughter%).
